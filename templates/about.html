Our Mission

The GDP Predictor is built to empower users with real-time insights into India's economic performance using intelligent predictions based on past data. We believe that understanding economic growth shouldn't be complex or inaccessible. That’s why we’ve created a tool that simplifies GDP forecasting for everyone — from students and researchers to policymakers and citizens.

Who We Are

We are a passionate team of data science students from IIT Patna, committed to making economic data more interactive and meaningful through technology. This project is part of our coursework to combine web development, machine learning, and real-world data interpretation into a practical, helpful application.

Features of This Project

  • Clean and user-friendly interface for GDP prediction
  • Data visualization powered by historical GDP trends
  • Sector-wise insights into India's economy
  • Real-time model-driven prediction for future years

Technologies Used

  • Python & Flask for backend development
  • HTML, CSS for frontend design
  • Pandas & Scikit-learn for data processing and machine learning
  • Chart.js or Recharts for data visualization
  • Render for deployment

Our Vision Ahead

Our goal is to continuously improve the prediction accuracy of the model by incorporating more real-time and granular data — such as inflation rates, sectoral indices, trade data, and more. We also aim to expand the scope beyond India to build comparative GDP insights globally.
